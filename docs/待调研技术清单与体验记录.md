# 待调研成熟技术清单与在线体验记录

> 本文档列出与"基于移动设备的人体建模与尺寸测量"项目相关的成熟技术，逐一记录官方网址、在线演示地址、注册/体验过程及评估结果。

## 一、2D人体关键点检测

### 1.1 MediaPipe Pose
- **类别**：2D/3D人体关键点检测
- **开发者**：Google
- **官方网址**：https://developers.google.com/mediapipe/solutions/vision/pose_landmarker
- **在线演示地址**：https://ai.google.dev/edge/mediapipe/solutions/vision/pose_landmarker/web_js
- **开源仓库**：https://github.com/google-ai-edge/mediapipe
- **技术特点**：33个3D关键点，支持浏览器/Android/iOS，CPU实时推理(30fps+)

### 1.2 OpenPose
- **类别**：2D/3D人体关键点检测
- **开发者**：CMU Perceptual Computing Lab
- **官方网址**：https://cmu-perceptual-computing-lab.github.io/
- **开源仓库**：https://github.com/CMU-Perceptual-Computing-Lab/openpose
- **技术特点**：135个关键点(含手部+面部+脚部)，精度业界标准，需GPU

### 1.3 MMPose
- **类别**：2D/3D人体关键点检测
- **开发者**：OpenMMLab (上海AI实验室)
- **官方网址**：https://mmpose.readthedocs.io/
- **开源仓库**：https://github.com/open-mmlab/mmpose
- **技术特点**：模型库最全，训练框架完善，中文文档好

## 二、三维人体重建（开源方案）

### 2.1 ECON (Explicit Clothed humans)
- **类别**：单张照片→3D穿衣服人体网格
- **开发者**：Max Planck Institute for Intelligent Systems
- **项目主页**：https://xiuyuliang.cn/econ/
- **开源仓库**：https://github.com/YuliangXiu/ECON
- **在线演示**：https://huggingface.co/spaces/Yuliang/ECON
- **技术特点**：秒级推理，显式网格输出，支持各种姿态

### 2.2 ICON (Implicit Clothed humans)
- **类别**：单张照片→3D穿衣服人体网格
- **开发者**：Max Planck Institute for Intelligent Systems
- **项目主页**：https://xiuyuliang.cn/icon/
- **开源仓库**：https://github.com/YuliangXiu/ICON
- **在线演示**：https://huggingface.co/spaces/Yuliang/ICON
- **技术特点**：SMPL先验约束，法向图辅助，极端姿态鲁棒

### 2.3 PIFuHD
- **类别**：单张照片→高精度3D人体网格
- **开发者**：Facebook Reality Labs / USC
- **项目主页**：https://shunsukesaito.github.io/PIFuHD/
- **开源仓库**：https://github.com/facebookresearch/pifuhd
- **在线演示**：无官方在线Demo，需本地部署
- **技术特点**：1024分辨率，细节丰富，推理较慢

### 2.4 SMPL / SMPL-X (参数化人体模型基座)
- **类别**：参数化裸体人体模型
- **开发者**：Max Planck Institute for Intelligent Systems
- **官方网址**：https://smpl.is.tue.mpg.de/
- **SMPL-X注册下载**：https://smpl-x.is.tue.mpg.de/
- ~~Meshcapade在线演示~~：https://meshcapade.com/
- **⚠ 重要更新**：Meshcapade已被Epic Games收购，在线平台于2026年4月18日关闭，SMPL演示不再可通过Web访问
- **技术特点**：业界标准参数化模型，所有重建方法的基座

### 2.5 PIXIE (全身3D人体重建)
- **类别**：单张照片→SMPL-X参数(身体+手+面部)
- **开发者**：Max Planck Institute for Intelligent Systems
- **项目主页**：https://pixie.is.tue.mpg.de/
- **开源仓库**：https://github.com/YadiraF/PIXIE
- **在线演示**：https://huggingface.co/spaces/YadiraF/PIXIE
- **技术特点**：端到端，同时输出身体/手部/面部参数

### 7.4 关键发现

1. **重大变化**：Meshcapade（SMPL官方演示平台）被Epic收购并关闭，意味着失去了最便捷的参数化人体Web体验入口。
2. **3DLOOK战略转向**：已从服装量体(YourFit)转向健康健身(FitXpress/GLP-1)，释放了服装量体SaaS市场空间。
3. **Virtusize最易体验**：提供11个真实品牌店的在线尺码推荐Demo，无需注册，是本项目产品交互设计的最佳参考。
4. **学术开源演示不靠谱**：HuggingFace Spaces的ECON/ICON/PIXIE受GPU配额和网络限制，不可用于产品演示，必须本地部署。

6. **商业模式验证**：B2B SaaS（3DLOOK/TrueFit/Virtusize）和C端订阅（ZOZOFIT）两条路径均被验证可行，B端ROI更清晰。
